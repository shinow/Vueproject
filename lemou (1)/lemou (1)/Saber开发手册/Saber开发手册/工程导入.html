前端安装

复制git地址

  1. 进入Saber项目首页：
  • 开源版：https://gitee.com/smallc/Saber
  • 商业版：https://gitee.ltd/blade/Saber
  1. 复制Saber的git地址
  • 开源版：
  • 商业版：
  1. 进入对应目录后克隆代码(windows可以用git bash客户端)

安装工程

  1. 安装yarn并配置淘宝源
> npm install -g yarn --registry=https://registry.npm.taobao.org

> yarn config set registry https://registry.npm.taobao.org -g
  1. 进入Saber根目录进行安装
> cd Saber

> yarn install

导入工程

  1. 打开WebStorm，点击File选择Open
  2. 找到对应目录的工程，并打开
  3. 看到如下界面则说明导入成功
